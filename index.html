tt.bot

A moderation and utility bot, made in JavaScript using the Eris library.

Apart from basic moderation and utility features, it features an emoji to picture feature to convert your emojis into a single large picture. On top of that, tt.bot provides extensions, a powerful way to interface with Discord using your own commands.
